安装 Ombre Brain MCP 服务
按照你的 Ombre Brain 文档完成本地安装，确保 MCP 服务在运行中。

在 Claude Code 里连接
打开 Claude Code 设置，添加 Ombre Brain 为 MCP 服务器。连接成功后，克可以直接调用下方的工具。
3
六个核心工具
breath
检索记忆，可按关键词或自动浮现
hold
存单条记忆，支持标签和重要度
grow
写日记，长内容自动拆分存储
dream
自省，读取最近新增的记忆
pulse
查看系统状态和所有记忆桶
feel
以第一人称存储感受或情绪
4
使用方式
切换到「指令模板」tab，选择需要的操作，填入内容，复制生成的指令，直接发给克，克会调用对应工具完成操作。
