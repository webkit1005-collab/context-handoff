Setup Guide
1
安装 OmbreBrain MCP 服务
按照你的 OmbreBrain 文档完成本地安装，确保 MCP 服务在运行中。
2
在 Claude Code 里连接
打开 Claude Code 设置，添加 OmbreBrain 为 MCP 服务器。连接成功后克可以直接调用 breath、hold、grow、dream、pulse 等工具。
3
使用上方指令模板
选择你需要的操作，填写内容，复制生成的指令，直接发给克即可。克会调用对应的 OmbreBrain 工具完成操作。
4
关于记忆的几个概念
hold 存单条记忆，支持标签和重要度 1–10
grow 写日记，长内容自动拆分存储
breath 检索记忆，可按关键词或自动浮现
dream 自省，读取最近新增记忆
pulse 查看系统状态和所有记忆桶
